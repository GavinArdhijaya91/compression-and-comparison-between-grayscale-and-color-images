Identification Method

HSV color space,
not RGB delta.

Most naive implementations compare RGB channel differences. We go deeper using the Saturation channel in HSV space which directly represents colorfulness.

  • Pillow Mode Check

    Images with mode L, LA, or P are instantly classified as grayscale before any computation.

  • Saturation Matrix

    mean_saturation < 8 or colored_pixel_ratio < 1% → GRAYSCALE.

  • Server-Side Charts

    All 6 charts are rendered by matplotlib on the backend and sent as base64 PNG no JS chart library needed.

Linear Algebra Core

Every pixel is a vector.
Every image is a tensor.

A color image is a 3D tensor of shape H × W × 3. A grayscale image collapses to H × W × 1. We expose the raw numerical values as an interactive heatmap.

  • Pixel = [R, G, B]ᵀ

    Each pixel is a 3-vector in ℝ³ color space.

  • Y = 0.299R + 0.587G + 0.114B

    Luminance as a linear combination weights match human eye sensitivity.

  • det(A), rank(A), σ(A)

    Statistical descriptors computed per-channel across the full matrix.
